💾 Backup & Dados

⚠️ IMPORTANTE - Como seus dados são salvos:
  • Os dados ficam salvos na memória do Chrome (não numa pasta)
  • Risco: Se limpar o Chrome ou formatar o PC → dados perdidos!
  • Solução: Faça backup semanal clicando em "Backup Completo"
  • Salve o arquivo na pasta do Google Drive para ter cópia na nuvem

☁️ Configurar Google Drive (Recomendado)

  1. Crie a pasta C:\Montgomery\Clinica\Backup\ no seu PC
  2. Instale o Google Drive Desktop (drive.google.com/drive/download)
  3. Sincronize essa pasta com o Drive
  4. No Chrome: Configurações → Downloads → Defina essa pasta como padrão
  5. Agora todo backup vai direto para a nuvem automaticamente!

📤 Exportar Dados

Faça backup completo ou exporte para Excel. Salve na pasta do Google Drive!

📥 Importar / Restaurar

Restaura um backup anterior. Substitui todos os dados atuais!

📱 Usar no Celular Android

  1. Salve o arquivo clinica.html no Google Drive
  2. No celular → abra o app Google Drive
  3. Encontre o arquivo → toque → "Abrir com Chrome"
  4. No Chrome → 3 pontinhos → "Adicionar à tela inicial"
  5. ✅ Aparece como app na tela do celular!

⚠️ Dados do celular ficam separados. Use Exportar/Importar para sincronizar.

🗑️ Limpar Dados

Irreversível! Faça backup antes.
